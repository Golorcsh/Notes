# Github使用技巧

1. in:  根据某个关键词进行检索
   
   关键词：
   - name
   - description 
   - readme
   
   使用方法：**需要检索内容** in:name 或 description 或 readme
2. 根据start 或fork 关键字查询
   
   单独使用语法:
   1. 数量范围：xxx关键词
stars :>=或者：<=
列如：查找stars数量大于等于2000的vue项目
vue stars:>=2000
   2. 区间范围:
XXX
关键词 fork: 数字1..数字2
列如：查找fork数量500到2000的springboot项目
springboot fork:500..2000
